# EXERCISE 6 - Fourier and Wavelet Transforms

This repository contains the Jupyter Notebook and the data needed for the exercise 6 of the Remote Sensing course.

The exercise is made of two parts:
- The first part (Exercise_6.ipynb) is about Fourier transform and their applications to image filtering.
- The second part (wavelet.ipynb) is about wavelet transforms, a specific type of Fourier transforms.

All instructions to execute the code can be found [here](https://github.com/ltelab/RS2026/blob/main/README.md).
